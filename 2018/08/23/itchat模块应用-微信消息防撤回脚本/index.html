关于itchat
itchat是一个开源的微信个人号接口，可以让你使用python调用微信，实现一些意想不到的功能。

2.注册消息方法
itchat将根据接收到的消息类型寻找对应的已注册的方法，如果一个消息类型没有对应的注册方法，该消息将会被舍弃，在运行过程中也可以动态注册方法，注册方式与结果不变。这段话直接理解有点费劲，看一下实例。


其中@itchat.msg_register(TEXT)表示，一旦接收到文字消息将调用下面的simple_reply(msg)注册方法，方法中的msg即是你接收到的消息内容，其中包含着大量信息，后面会详细介绍。

simple_reply(msg)的功能是，当你收到文字消息时，文件传输助手会给你发一条“你收到一条文字消息”作为提醒，同时会回复给对方“我已收到内容为-xxxx-的消息”，msg[‘Text’]是对方给你发送的消息内容。

此处的toUserName不是微信号，也不是昵称或者备注，它是一个位数很长的一串字符。

3.关于微信消息
3.1消息类型
微信消息多种多样，除了文字、语音、图片、视频之外，还有其他多种多样的消息。所有的消息类型如下表：

参数 类型 Text键值
TEXT 文本 文本内容（文字消息）
MAP 地图 位置文本（位置分享）
CARD 名片 推荐人字典（推荐人名片）
SHARING 分享 分享名称（分享的音乐或者文章等）
PICTURE 图片、表情包 下载方法
RECORDING 语音 下载方法
ATTACHMENT 附件 下载方法
VIDEO 小视频 下载方法
FRIENDS 好友邀请 添加好友所需的参数
SYSTEM 系统消息 比如群聊的UserName组成的列表
NOTE 通知消息 通知文本
