但是这个网址很不美观，至少逼格不够高，要弄就弄世界上独一无二的网站，如果有这种想法，你可以去买个域名。


二.绑定个人域名
 1.购买域名
可以去万网，也可以去腾讯云平台购买（前面的万网是属于阿里云的）,购买的过程就是一系列注册认证的过程，略去。

2.配置DNS地址
在哪个平台购买的域名就在哪个平台的控制台修改，将DNS改为DNSPod的免费DNS地址：
f1g1ns1.dnspod.net
f1g1ns2.dnspod.net



3.域名解析
注册一个DNSPod账号，登录之后把我们新注册的域名加进去，在域名解析界面添加3条记录

@ A 192.30.252.153
@ A 192.30.252.154
www CNAME username.github.io.


改完后的结果大概是这样的


4.添加CNAME文件
用文本编辑器新建一个名为CNAME的文本文件，内容仅仅是你的域名地址，即 www.xxxxxx.com/net 啥的。添加完内容后删去后缀名！！！将该文件放入本地博客的source文件夹里面，然后一下命令更新到GitHub。（每次修改完本地博客文件后最好都要输入以下命令）
